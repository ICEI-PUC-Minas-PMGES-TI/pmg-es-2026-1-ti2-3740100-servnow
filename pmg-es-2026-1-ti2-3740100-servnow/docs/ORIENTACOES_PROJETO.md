## Padrões para commit

| Prefixo | Uso |
|---------|-----|
| `feat:` | Nova funcionalidade |
| `fix:` | Correção de bug |
| `refactor:` | Mudança interna sem alterar comportamento |
| `chore:` | Config, limpeza, etc. |
| `docs:` | Documentação |
| `test:` | Testes |
| `style:` | Formatação (indentação, etc.) |
